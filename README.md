# rtf

## Dependencies

- CUDA
- Vulkan SDK
- xmake
- sed (for Windows)

## Build

- At proj/static_render:

```bash
xmake build

```

## Assets

- Download assets from github release
- Move assets to proj/static_render/assets

## Run

- At proj/static_render:

```bash
./build/static_render
```

or

```bash
xmake run -w .
```
